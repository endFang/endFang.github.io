游戏设定的背后 月亮是魔女题材的一个重点。几天看机核文章讨论了 《猎天使魔女》中的魔女形象溯源。一条评论说神谷英树没想那么多。我不同意。魔女这个概念从诞生那一刻开始就已经开始利文化沉淀，从魔鬼理论，魔鬼契约，魔鬼印记，到魔女审判，《女巫之锤》和猎巫运动。这些历史与文化都和魔女紧紧联系在一起。也许神谷英树没有做非常深度的调查研究，但是当他创造了 Bayonetta 的时候，这些背景都会或多或少的随着“魔女”融入进了游戏中。 当我看到“抹茶”，我会首先想到“绿色”，“甜点”，“日本”。稍微走远一点，“茶杯“，”茶道“，”苦涩”。再远一点，“中日文化“，”文化输出“，”唐朝“，”陶瓷烧制“。最先看到的是表象，往后走才是背景和溯源。 FFXIV: Mentor Roulette Mentor will party with new/veteran players to complete old content. 从 Soylent 到杰克苏

Before 在感恩节放假之前，我就已经预见了我不会出门，不会抢购，宅在家里的景象。因此提前预购了四箱 Soylent 作为下一个月的食物。 Cafe 系列添加了咖啡因和茶氨酸，作为早餐。Soylent 系列作为晚餐。家里还有剩余的食材，每天做一顿午饭不成问题。 Drink Soylent 已经推出了各种各样的口味。Cacao 和 Vanilla 的最好喝，Original 和 Coffiest 次之，最难喝的是 Chai。 每一瓶可以满足每日所需营养的 20%。 喝完之后的感受就是觉得嘴里没味。肚子是饱了，可是总觉得缺点什么。我并不执着于美食，所以可能是缺少了 “进食” 这个动作？ Eat 家里的豆腐还挺多。便试做了“二十四桥明月夜”的变体版：培根蒸豆腐。第一次尝试：培根出油太多导致豆腐特别油腻。 但是豆腐怎么做都好吃，直接蘸着酱油也好吃。我就喜欢吃豆腐。此处故意双关。我还喜欢吃菠萝披萨，绝对不会有人抢。 剩饭可以做成炒饭。我喜欢炒饭。此处依旧故意双关。从 “侠·饭” 第一集就学到了沙丁鱼罐头的妙用，因此把平常用的植物油换成了罐头里的油。真好吃。配上豆腐也好吃。 从 ”基本演绎法“ 中学到了福尔摩斯的炒鸡蛋。四个蛋加上一点点奶油，还有糖和盐。Amanda 和韩尚宫娘娘说得很对，味道之间会互相调和，加一点点盐可以使得糖的甜味更明显。撒一点点黑胡椒也很好吃。 Play 狂战传说打折，买了下来。感想是：我再也不会玩传说系列了。 人物塑造得过分中二。“纯洁小花被背叛后化身反社会分子然后与一群偏执狂拯救世界”。 传说系列的一个特点就是会蹦出各种各样的对话，包括对目前剧情的看法，人物之间的互动，以及为未来剧情埋下伏笔。但是大部分都是冗长而且繁琐。我又不是来看小说的，可能我不再喜欢 RPG 了吧。 非常频繁的跑稍微复杂了点的地图。这个是很麻烦的。 我发现了我的身体会自动判定一个游戏是否好玩：凡是我喜欢的游戏，我通常都可以坚持一两个小时，比如 FFXIV，FFXV，Bayoneta，DmC，Elder Scroll，Neverwinter，Dragon Age，Salt and Sanctuary。我并不喜欢 FFXV，我是 FFXV黑。凡是我不怎么喜欢的，半个小时内就会觉得头晕，比如 Witcher，ToS，ToB，Dragon Dogma。 Read 发现了一本 BL 小说。《说好要杀死我的人都爱上了我》。 一看名字就是杰克苏的甜文。我比较喜欢刀子。他人怀糖罐，我有笔如刀。 内容可以概括为”霸道总裁爱上我“ x4 以及”撸猫才是正经事“。 After 荒废了时光。漫画和游戏世界真有趣。 豆腐真好吃，未来一年可能会吃很多豆腐。

4.0 之后，4.1 之前

还有几天就升 4.1。惯例写总结。 单手挡住了所有玩家的 Raubahn 4.0 Early Access 刚好在我结束最后一个 Final 的晚上进行升级。也就是说考完试第二天 4.0 的服务器也会上线。特意在考后买了鸡肉卷，橙汁，还有大桶水。两天两夜的游戏马拉松准备完毕。那天服用了褪黑素后八点就睡了。凌晨两三点手机闹铃一响我就蹦下床，一边登陆服务器一边刷牙洗脸。 Alphy 又开始忽悠我去拯救 Ala Mhigo 了。我并不想去，Alphy 说对我很失望。我还能怎么办。 整个游戏体验非常顺利。但是 chat log 蹦出来一条公告，大意是：duty finder 现在出了 bug，所以无法下副本，连 instance 也无法进入。当时我并不在意，认为这只是一个小错误，何况距离副本还远着呢。但是任务 Best Served with Cold Steel 的 solo duty 就把我拦下了。其它玩家也被拦下了。不管怎么做，系统永远都会跳出一个提示： A system error occurred during event movement. 很快这个任务点就出现了一条队伍，却并没有解决问题。chat log 充斥着不耐烦的抱怨，劝服其他人排队的安慰，还有一个非常特别的 “queue log”。reddit 上也开始出现 “Raubahn EX Guide” 或者 “I’m very disappointed” 的帖子。一直等到太阳升起，这个 bug 还是没有被修复。此时更多的玩家来到了这里，原本就在等待的人的火气也变得很高了。反正我是在太阳升起后就去睡觉了，原本的马拉松也推迟。 差不多快一周后， SE 发声明说这是 DDoS 攻击。 How very convenient. Y’shtola bots 遍地走的 PVP 剧情推进不了我去升级，毕竟 PVP 现在也给 XP 了。而且 PVP 也独立于 PVE，技能和能力大幅减少，所有 stats 全部重设。这个设定挺聪明的。PVP 独立后就可以放开束缚随意修正，因为不用担心会影响到 PVE。...
