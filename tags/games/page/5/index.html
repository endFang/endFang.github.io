友情设定：溪谷恩

Intro 溪谷恩是在和 Y 闲扯后遐想出来的 Boss。 最初的灵感来自 “You! You! Delay! ” 这个由于没有按时出发而脑洞出来的 delay 技能。 技能名：You! You! Delay! 效果：指定目标无法按计划出发。比如即使提早到了机场也会因为证件不合格而被机场人员质疑，最终为了保险只能离开重新办理证件。 溪谷恩的初版设定是能控制时间的有着“骚气外形”的大 boss。背景乐“诡异暗黑”或者“轻灵哀婉”。战斗结束后还掉落会唱战斗 BGM 的发条小鸟和“彩蛋物品”。特定情况下还会出来一个“支线用来补充背景设定”。“小技能减速，大技能是延长CD”。 引号里的内容是 Y 自己要求的。 Lore 的部分是我用来写 Era 的第二版背景设定。Game 则是脑洞出来的初版游戏设计。 Lore Siguon, the Lady of Bond, the Mysterious Ribbon 溪谷恩，纽带女士，神秘缎带 溪谷恩是代表万物联系的女神，能看透所有人事物的前后联系和因果。最复杂的问题在溪谷恩的面前也能被一一解开。她能修复破裂的感情，维护人们的尊严，也能挑拨最亲密的爱人，令他们老死不相往来。她的信徒认为女神的祝福会让他们生活在平静中，避免无谓的争端。惹怒女神则会使得万事不顺，一点小小的摩擦就会导致大麻烦。 由于其特殊的神力，溪谷恩能知晓许多她并不愿意知道的事情，因此她不会轻易动用力量。 纹章是一条代表连接和因果的缎带。 在众神纪元的时候，溪谷恩是抵达“下层世界”的神祗之一，但是她并没有与“下层世界”的居民结合留下任何后代。 戮神战役开始后，她首先看透了白铁盟誓和灾祸神之间的奇怪纽带，随后提醒了银狼神和主神。 战役结束后，溪谷恩参与了神对“下层世界”的审判。由于其看透万物联系的力量，是一位重要的证人。 Game 以下为参考 FF14 制作出来的游戏内技能： 前因后果 Cause and Effect 点名两个 DPS，分别获得 Cause/前因 和 Effect/后果 两个不同的 debuff 并且开始倒数。两位玩家同时被缎带束缚无法自由移动但是可以攻击。必须在 buff 倒数内先打破 Cause 的缎带束缚才能解除 Effect 的缎带束缚。否则 Cause 和 Effect 会依次爆炸造成中等 AoE 伤害。...
