Unity Shader Overview

Posted on 2019-11-30 Edited on 2020-11-30

Thanks to these posts and blogs:
Michael Sanders
Catike Coding
Alan Zucconi

Unity has 3 types of shader: surface shader, fragment and vertex shader, and the obsolete fixed function shader. They all have the same anatomy.

Catlike Coding

One shader can have multiple SubShader . They contain the instructions for the GPU. Unity execute one by one until it find the compatible one.
2D properties = textures. bumb indicates that the texture is used as a normal map. Vector and Color always have 4 elements.
Define the properties variables in the SubShader before using them.
Tags are telling Unity certain properties of the shader. “Queue” tells the RenderType and controls the render order. Queue accept positive integer too. The smaller, the sooner is drawn.

  • Background (1000): background and skyboxes
  • Geometry (2000): default label used for most solid objects
  • Transparent (3000): materials with transparent properties
  • Overlay (4000): effects used as lens flares, GUI elements and texts.

Relative orders are acceptable, such as Background+2. Messing up the Queue might cause nasty situations.

CGPROGRAM and ENDCG are keywords to separate the shader program from other statements. They are old design decisions that made sense once, now we’re still stuck with the backwards compatibility.

ZTest

An object from Transparent doesn’t always appear above on object from Geometry because GPU performs ZTest which stops hidden pixels from being drawn.
GPU uses an extra buffer to store the depth of the object drawn in that pixel. Any further away pixel is discarded.

Surface Shader

Use surface shader if the material is affected by lights realistically. Surface shader also hides the calculation of how light isrefelcted.

Vertex and fragment shader

UnityCG.cginc is one of the shader files bundled with Unity. It includes some other essentials files and contains generic functionality.

UnityCG contains UnityInstancing and UnityShaderVariables, UnityInstancing depends on UnityShaderVariables. UnityShaderVariables includes HLSLsupport.
UnityShaderVariables defines shader variables like transformation, camera, and light data.
HLSLSupport helps us to use the same code, no matter which platform we’re targeting.
UnityInstancing is specifically for instancing support.
Include a file more than once will make the content copied more than once. It might lead to compiler errors for duplicate definitions.

pragma is to issue special compiler directives.

Geometry model is firstly passed thru the vert to alter the vertices. Then individual traingles are passed thru frag to decide the final RGB color.
Useful for 2D effects, post-processing and special 3D effects.

*Each shader consists of a list of subshaders.

SV_POSITION: SV stands for the System Value, POSITION is the final vertex position
SV_TARGET means the default shader target. SV_TARGET1 or 2 indicates the additional render target. 0 is the default.

Any other semantics (not SV) are defined completely by the user is to match the outputs from one stage of the pipeline to the inputs of another stage.

swizzle operations: flexible access a single or more component of a vector. e.g. .x, .xy, .yx

Texture mapping: to change the tiling and offset, use the TRANSFORM_TEX macro. It will multiply the tilling and add offset. It’s not need to multiply by _Main_ST.
ST = Scale and Translation, it’s an old terminology.

Texels: pixels of a texture
Filter Mode:

  • Point: the nearest texel is used when a texture is sampled at some UV coordinates. (blocky appearance)
  • Bilinear: interpolation between 2 texels. Good for texel density is less than the display pixel but not the other way around, because adjacent pixels will end up with sampels taht are mot than one texel apart.

放假后从早玩到晚的古剑奇谭3

Posted on 2019-10-04

Intro

抓到了打折的尾巴。Nice。天朝的 PC 端游戏落后果然是真的。我不是说古剑奇谭3是个糟糕的游戏，但是在比较之下差异就会显现出来。

石化咒与无声咒

我第一个注意到的地方就是剧情对话中人物的脸上都是没有表情的。我可以看见人物的眉毛和嘴巴在动，但是我还是觉得他们的表情无限接近于零。跟个石头一样。人物的动作衔接也并不是十分自然，尤其是在过场剧情中。又不是每一个场景都是龙宫皮影戏。

顺带一提，龙宫是整个游戏中创意表达最优秀的地方。过场剧情由皮影戏和戏曲腔调来展现。战斗也靠着“影”从 3D 变成 2D 横轴。

配乐很棒，比方说无名之地的背景乐中并没有用急促紧张的节奏或者尖锐的乐器来表现一个鬼地，而是用了缓慢忧伤的调子，其中还混入了鬼哭声。
但是嘴巴动作和配音对不上，而且很多战斗的剧情中的音效都像被施了无声咒一样：那只我叫不出名字的鸟妖拍打翅膀没有声音，獍妖的吼叫没有发声，北洛做出一个反人类动作也没有声音。我觉得画面和音乐已经是好的了，但没有声效就显得所做的努力全白费了。

你把任务日志藏在哪里了

我所见过的垃圾的 RPG 任务系统之一。任务有主线，支线还有场景。一次只能追踪一个主线和一个支线/场景。场景还要到了相对应的地图才会出来。任务记录和进程不能任意快速查看，全靠地图界面右边侧栏。你好歹给我个便捷的窗口显示一下我接了哪些任务以及做到哪个部分了好吗？那个青桐剑和壁画的任务还是我偶然进了剑炉和无名之地触发了对话才想起来还有这么个事情。侠义榜上有很多都是要制作或者收集的物品，收集到了任务物品也不会有提升说已经收集了多少个，我还得在物品栏里面一个个翻啊？
还有主菜单每次都要加载，等那个圆圈转一下。急死我了。每次退出子菜单，默认选项也会退回到“人物”还是“队伍”，烦人。

不上不下的战斗

一开始出现的选择控制模式出现了 “ACT”，我以为这是靠组合 combo 来战斗的游戏，结果是靠积攒元气来释放技能的类型。可是元气积攒的很慢…… 为了要提高连击数来发动星蕴的效果（30 以上）我就必须靠不中断攻击，很多爬行动物的攻击必须靠技能打断，不然会被打。所以我陷入了“一格元气到手 —— 爬行动物要发招了 —— 发动技能 —— 打断爬行动物 —— 元气清零” 的死循环中，很少使用需要两格元气的技能。就算放弃连击数，靠契约生物和技能打，战斗还是很枯燥。普通的轻重攻击组合的 combo 效果并不突出，剑技能和召唤还有队友的协作必须等，所以成了一直在磨血的战斗。精力可以用来防御，闪避和冲刺，但是如果防御成功，短时间内闪避和冲刺就会减少，而且防御反击的成功率都我来说连 50% 都不到。一场打下来非常累。再加上每次都见到一样的敌人模型和攻击方式。第一次到了经天轮中后我看见一地的爬行动物就完全放弃了战斗，一路跑到了山顶上。

另外，为什么我不能控制岑缨和云无月！除了能打 combo 的鬼泣，DAI 中骑士魔法的剑，和 rapier，在所有武器中，剑是我不想用的武器之一。我都能给他们加技能点数了就不能让我控制他们战斗吗？

人人都爱千秋戏

是个 NPC 就爱玩千秋戏。侠义榜上也有很多挑战。是不是在古剑的世界中千秋戏有围棋的地位？玩得厉害还能成为国手享无上荣耀？我每次都抽到需要欧阳少恭的牌，但是欧阳少恭宗总是不出现，气死我了。
千秋戏给我的感觉和花札牌很像，都是靠做出组合来得分。越难做的组合得分越高。我做过的最高点是“古剑奇谭”，就莫名其妙就收集到了所有的剑，然后分数点蹭的一下就跑到了100以上。
这个比 FF14 的 Triple Triad 要老实本分得多。TT 经常玩到最后出现连续 combo 把玩家耍得团团转，千秋戏还会给你提示。
感觉能出精美周边。出了我就考虑买。

似乎要把家园玩塌了

抛开古剑不说，所有的国产游戏都一定要有一个家园是怎么回事？是因为历史上自给自足的的小农经济带来的影响吗？
不玩家园活不下去。除去家具和 customization，家园系统看似方便，实际上是被镶嵌的游戏体验中了。侠义榜上的 10 个鹿活草我就真的不想跑到野外去收集。野外地图很大但是能探索和收集的地方很少。到了中期所有的装备都只能从家园的建造中获得。建造就要靠收集材料和工匠。材料靠契约种族和种植，这些雇佣工人也要靠材料和烹饪存活。这个完美的自给自足的小农经济…… 那个洒水声音的最后有点上不了台面。

梦能解释一切

但玩家不是弗洛伊德。有一部分剧情是靠着梦来解释的。我不知道北洛到底在剧情的发展中知道些什么，但是他和缙云之间的联系被解释得很粗糙。XIV 中 Y’shtola 失去视力后没有明说，但是 Matoya 问她为什么眼中失去了光彩后玩家都知道 Y’shtola 已经确实不能靠眼睛来看了。古剑3中很多事情都是靠“因为在梦中见过了”来解释，你说什么就是什么吧，反正我是没看到。我记得很清楚，从梦中的巫之国那里开始，北洛就突然啥都知道了。特别多的名次冒了出来，我是没有听懂。和 王国之心3 的叙事有得一拼。

还有我讨厌控制缙云。

Conclusion

游戏中的剧情并不多，但是我特别感激这个游戏的主题是“inheritance and legacy”而不是 love。从远古时期就被流传至今并且不断发展的知识与文化，和由一代又一代人所书写的历史，这些听上去就比 love 大气得多。评分上很多人都在夸，但是以我的标准还达不到十分优秀能让我发出“唉”的一声的程度，而是在值得一玩的程度。

Shader Basic

Posted on 2019-09-23 Edited on 2020-11-30

Basics

  • shader language has a single main that returns color at the end
  • final pixel color is assigned to gl_FragColor
  • C-flavored language has built in variable, function, type
  • preprocessor macros (pre-compiled)
    • #define global variable
    • #ifdef and #endif
  • lower precision → faster rendering, less quality
    • precision medium float
    • precision lowp float
    • precision highp float
  • GLSL specs doesn’t guarantee auto-cast
    • vec4(1, 0, 0, 1) will raise error, because it’s not float
  • multiple ways of constructing vec4
    • vec4 color = vec4(vec3(1., 1., 0.), 1.);

Uniform & Varying

  • GPU has multiple threads calculating a portion of the color, each parallel thread is blind to others. The input sent from CPU to GPU are equal and read only to all the threads. These are uniforms.
  • supported type: float, vec2, vec3, vec4, mat2, mat3, mat4, sampler2D, samplerCube
  • uniforms are defined on the top of the shader after the point precision assigning.
  • GPU hardware has accelerated angle, trigonometric, and exponential functions
    • sin(), cos(), tan(), asin(), acos(), atan(), pow(), exp(), log() …
  • gl_FragColor is the default input. It holds the pixel coordinates of the pixels or screen of fragment that the active thread is working on. It’s not a uniform, because threads have it differently, it’s a varying. A varying variable contains data shared from vertex shader to a fragment shader.

5.0 的熬夜

Posted on 2019-09-02 Edited on 2019-09-22

我根本就没有睡着。我是第一批登陆服务器的玩家。

AST

很多人都不喜欢 AST 的技能修改：从六种不同的 buff 到 只增加伤害。对我来说其实并没有改变多少，以前的六种不同的 buff 最终目的都是增加伤害，比如 BLM 是减少咏唱时间 的 Arrow，DPS 是增加伤害的 Balance，Tank 是减少受到的伤害的 Bole。现在每一张卡片的效果都是增加伤害，只是对应了不同的 role 而已；而且以前大部分人都只想要抽到 Balance，现在的目的则是抽到不同的卡片来收集 seals。一旦有三种不同的 Seals 就可以使用 Divination 来进一步增加伤害。不过是从 Balance 变成 Seal 而已。没有很大的区别。
不过由于 sects 缺少了 healing buff，以及 Largess 的消失，所以在 leveling dungeon 中很难处理 big pull。就我的体验来说，WHM 是所有 healer 中最完美的，既能 burst healing 又能使用 shielding。SCH 的 新仙女 Seraph 的用处并不明显。我对于 SMN 和 SCH 的新的召唤唯一不满的地方就是剧情上 和 lore 上并没有解释为什么会出现新的召唤技能。现在每一个 Job 都只有一个 80 级的 Job Quest，剧情非常少。5.0 集中在了 Role Quest 上。玩家通过升级不同的 role 来回顾 WoD 的冒险。

5.01 中修正了 AST 的治疗效果。万岁。AST 的裙子终于可以在男性身上出现了。终于可以穿裙子了。

Plot

剧情上跌宕起伏。3.4 的时候我们就已经得知 The First 中 Light 与 Darkness 的不平衡导致了世界即将会面。 WoD 被 Hydaelyn 所附身的 Minfilia 带回 The First，成功阻止了 Flood of Light，但也不过是权宜之计。与此同时，在另一条时间线上，The Source 也因为元素失衡和 Empire 使用 Black Rose 导致了世界大乱。G’raha Tia 被唤醒，与 Garlond Ironwork Company 一起试图回到过去来阻止末日。他们研究了 Crystal Tower，Alexander，和 Omega 后造出了 Tycoon，可惜并没有成功。牺牲了许多人之后，G’raha 和 Crystal Tower 一起被传送到了过去，不过有 100 年的误差。G’raha 的目标原本是将 The Source 的 WoL 传送到 The First，但是多次失误只传送了 Scions。
The Source 因为有太多的 Light，就连在夜晚天空也是一片光芒。唯一的消除 Ligth 的方法是击败 Light Warden，吸收它们的 Light。另一边，Minfilia 在 The First 不断转世成新的女孩来对抗 Sin Eaters。每一个女孩都有着金色的头发和清澈的蓝宝石眼睛。中间 Emet Selch 也加入，揭露了 Hydaelyn 和 Zodiark 实际上都是 Primals。最终就连 WoL 也无法承受如此多的 Light，G’raha 秘密计划在最后一刻要自己吸收所有的 Light 再将自己传送。螳螂捕蝉，黄雀在后，Emet Selch 用手枪伤了 G’raha，并且将 WoL 引导到了 复制的 Amaurot 中。决战后，WoL 用尽 Light 击败了 Emet Selch。

G’raha 大有取代 Haurchefant 的趋势。我是开玩笑的。

5.0 中并没有新的 Primal 出现，Trial 中加入的是 fairy queen：Titania，和 sin eater：Innonce。Titania 是 fae 中皇后的称呼。玩家遇到的第一位 fae 则是 Feo Ul。她作为 The First 和 The Source 的桥梁帮助玩家呼唤 retainers 和传递消息。Feo Ul 个性强势，配音完美。她也在玩家击败 Titania 之后代替玩家成为了新的 Titania。Fae 也不限于 fairy 或者 pixeis，还有长得像青蛙的 Fuath，FF 中经常出现的 Nu Mou，以及漆黑的 Amaro。

I like the idea of paying digitally, but I prefer not to use it. After three years of swiping cards and Apple Pay, I realized that I am less sensitive to the numbers. Digital payment tells the number of money I have and spend, but it doesn’t give a clear idea of how much I have and spend. The action of taking out cash and hand it to the cashier tells me that I used 5 dollars and I have 5 dollars less, but I can only see the “-5.00” on screen after I swipe the card. My parents praise highly on digital payment and laugh at me when I use the cash. They didn’t and wouldn’t understand even I tried to explain. They also don’t understand why the USA people don’t use an APP to make the payment. “American’s technology is not that developed, they still use card and cash.” No, technology is not the reason. It is the banks’ benefits and third-party payment agreement. But I prefer not to explain these, I know they wouldn’t listen.

Hepburn

I’ve been watch Hepburn’s old movie. The old movie is funny and romantic. That’s what Mr. David said. It’s quite common that a cunning fox man plays around with the girl, and probably lies and reveals his identity at the end. The girl never had experience before she meets this man, or it looks like she never had. Nicole Bonnet in How To Steal A Million loses her mind after Simon Dermott kissed her. The “oh what a man” face is funny and exaggerated. Nobody does that in real life. At least I haven’t seen anyone has such a funny face. Same to Charade. Ariane in Love in The Afternoon behave more normally. Mr. Frank attracts her and distracters her from her daily life. I also like what Eliza did to Professor Higgins in My Fair Lady. She keeps yelling “I’M A GOOD GIRL” and cries for help when the maids try to bath her. Well trained by the professor, Eliza successfully fools Karpathy the phonetics expert. When all the praise goes to Higgins, Eliza’s hard work is not recognized. Professor’s indifference to her future also irritates her. She throws slippers on Professor’s face. Nice one. When a girl can’t help fall in love with a man in the old movie, I tend to roll my eyes. Sabrina in Sabrina acts in the same way at the beginning, but then she stands up and becomes more confident and become helpless after 10 mins.

The heartless life likes to mock us. It made us meet each other, yet give us so little time.

His brown-ish yellow eyes are beautiful. I can see me reflected in his eyes. His hands are bigger than mine and can easily hold mine. He likes kissing. He said the trees standing in the other side would gradually turns to yellow and red in the fall and eventually falls on the ground. He wore cologne that made me think of him all the time. He looks like a ghost in the dark. He fell in light sleep quickly. His breath was like breeze.

I saw Mississippi River. The riverside was muddy. The sunlight burned the ground and we stood in the shadow. I remember the iced tea, the LED lights, and the thunder.

The morning was humid and warm. I walked on the wrong direction but made it in the last minute. I saw people gathered near the door. A madam smiled and nodded to me. I smiled back and sit in the last row. I saw him playing and singing on the stage with the choir. I couldn’t stop to look at him. I remember the trees, squirrels, muddy road, grass, and the mushroom. People were talking on the boat. We hold our hands when he was driving. The lemon soft ice cream tasted like watermelon in my first bite. I remember the angle and the demon. I felt his breath again.

He kissed me in the morning and woke me up. I saw the train ran to the platform. Chicago was windy indeed. I saw the lake that looked like the sea. I saw the exhibition and listened the music. The peanut butter cookie was tasty. I saw the city skyline and was terrified on the wheel. He hold me and kissed me. I remember the caramel and cheese popcorns, chicken dumplings, sparkling water and green tea.

We woke up at five. The cologne was on my shirt. I remembered what he said before I leave. I shall kept it in my mind.

Forward XVI

Posted on 2019-07-19 Edited on 2019-07-26

7.8

Do it when you could. 这就是 Forward 现在所教给我的。出发的时候我很难过，现在旅程开始朝着终点加速，我还是很难过。这六年来我一直在说“光阴似箭日月如梭”，这一次是真的时间飞逝了。“欢乐的时光总是过的特别快。”我十分确定这种感觉以后一定还会有。

最早知道 California Zephyr 是 2015 年的春天，当时读到了关于火车上 Game Jam 的新闻。一群人在火车上组成队伍，两天内制作出一部游戏作品。到了终点后就开始展示队伍的成果。

我预订了一间 rommette。由于是一个人，可以独享两个座位和床。要经历两个晚上还是不要坐在一般的椅子上了。窗户很大，有窗帘。下午的时候太阳会直射进车厢。
Roomette 包括了所有的 meals。我发现 dinning cart 里面大部分乘客都是老人。可能是年轻人都想着自驾或者飞行吧。我点了一份鸡胸肉。一盘下去我觉得我真的太饱了。同桌的人还点了点心。我彻底放弃，根本吃不下。我想起了很久很久以前唐老师不准我在饭桌上浪费食物。

一开始的窗外风景大多是田地，大草坪和小树林。偶尔能看见小镇或者工厂。
老实说我很希望我的对面能坐着一个人。我决定在到达终点后再去一次洛杉矶的 Universal，以及去格里菲斯天文台。我想再看看。

因为是在火车上，所以试着把 Grindr 加上了照片。反正我是在异国一辆行驶的火车上，也不会有什么严重的后果。顶多就是很倒霉地让一个反社会分子把我的照片 PS 后 po 到网上最终成为 meme 永远被人笑话或者被 serial killer 看上然后被谋杀。考虑到很多 Grindr 用户都用了照片，这种事情发生的几率还是很小很小。顶多就是被笑上长得丑陋。实际上还是有人搭话，我借这个机会练习如何在谈话中夸人。就比如说有人的照片是小腹，我就夸他的肚脐很好看。
我真的没什么夸人的能力。
夸了几轮之后，我觉得我掌握了聊天的技巧。比如有人的身上有纹身我会夸纹身很酷。总之就是根据照片来找一个点。光说 hi 和 hello 可么啥用。然后我被疑似骗子的 sugar daddy 搭话了。对方说想找一个 sugar babby，每周给 500 零花钱。那一个月不就有 2000 了？哪有这种好事。尤其是对方能看见我离他有多远的情况下。为什么要找一个只能发消息不能见面的 baby 呢。脑子坏了吗？
Grindr 毕竟早就沦为约炮工具了。练习完夸人之后就要练习拒绝。夸完就跑，听上去好渣。好吧，我在火车上本身就是个拒绝，都不要练习瞎编理由。但有一个人很特别，就算我跟他说了我在火车上之后他还是继续和我聊天，也问我会不会再去伊利亚诺斯中部，他愿意见我。他刚好是我喜欢的类型。好吧，这可能是花招，但是从没经历过这种事情的我懵了。就像 Minfilia 被 Thancred 摸头。这可能是我夸完就跑的报应吧。
气死我了。

7.9

昨晚 Nebraska 有风暴，铁轨上覆盖了泥沙，需要清洗才能行车。所以从 2:30 am 开始火车就停在了 Lincoln 站台。半夜我就觉得好冷，但我以为是 AC 太强了。早餐也从 7 am 才开始供应。一直到 10:30am 火车才开始行使，这之前我一直用热点玩 XIV。
窗外的景色总是一样的：农场和草坪，偶尔的小镇。没有什么特别的景象。下午开始洗脑循环 Satisfied。
Surge 上还是没有 match。Grindr 上的风向倒是开始变化了，搭讪的人已经全部变成了 daddies。我都不知道该从哪里了开始夸，都是上来就是问要不要做他的 sugar babby。我觉得大多数都是 scammer。放着附近的真人不着要却来找虚拟世界的人，还只要 sex chat 和 naughty pics。我还从一位用户的 profile 上学到了怼人句式：“Save your dick pics. You’re on Grindr, I know you have one.”老实说我觉得我已经没有在练习夸人，而是在观察随着不同地区不同人是如何 hook up 的。

7.10

我遇见了 Henry 和 Lisa。又是一对 50 周年的 couple。Henry 会说法语和阿拉伯语，他鼓励和帮助学生们去其它的国家见识和学习不同的文化。因此他很好奇我对美国的看法。
火车延迟了很多次。昨天在 Lincoln 停了 10 个小时，今天至少在四个不同的地方听了 1 个小时。列车长告诉我火车已经晚点了，所以明天早上会直接用巴士把我们送到目的地。我原本还在想着要不要去 San Fransisco，现在我想干脆直接去洛杉矶得了。

Grindr 上今天啥事都没发生。要么是 Colorado 对亚洲人毫无兴趣，要么就是对我毫无兴趣了。说不失望是假的，总是会希望有人会对自己感兴趣。Surge 倒是有一个 power like。我勒个擦。我没敢回复。作为一个在火车上用约炮软件练习聊天和夸人，聊完就跑的渣来说不知道要怎么回复。

晚上的晚餐是免费的，对我这种在 sleeper cart 的乘客来说并没有什么改变，反正我的 meals 原本就是免费的。Sleeper cart 的 Rommette 还真是一个迷你房间。两张软垫座位可以拉伸成一张床，小桌板可以收缩，有小垃圾桶，镜子，衣架和小衣柜。下层车厢有浴室可以洗澡。晚上我去 Lounge cart 拿水的时候发现许多 Family bedroom 都空了，我溜进去看了看，对于车厢来说还是挺宽敞的，可以放下两张火车上的床，还有能伸腿的空间，并且自带了洗漱台和浴室。我突然觉得，如果只是一个人住的话，这种超迷你型的公寓我也是可以接受的。当然基本大小是 Family Bedroom 那么大，两张椅子拼成床，衣柜和置物架可以放一边。只是没有厨房。要真有这种迷你型公寓，厨房不要也罢。但还是会想要一个小冰箱。

下午的时候又搭巴士又搭火车的终于到了 Emeryvile。我在火车上开始写新的小短篇，真是文思泉涌。

Elric 家是上个世纪 20 年代时的住宅，他把房子翻修了一遍。家具十分特别。原本桌子的四脚全都用了水管代替，桌面上全是涂鸦。灯泡带着一点蒸汽朋克的风格。
