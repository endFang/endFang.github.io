Forward IV

Posted on 2019-06-19 Edited on 2019-06-20

难得的起早了。昨天还是晴天的西雅图在早上就开始下雨了。我忘了带 Nero 外套，决定一整天都要尽量呆在室内。

去了一家叫做 Biscuit Bitch 的店吃早餐。我在结账的时候弄错了店员，把订单纸递给了负责饮料的人。她们可能以为我并不会说英文或者英文不好，所以当着我的面说“Yeah, I don’t know why he gives me the order.” 我想了想，回了一句：“Because you are wearing the apron and the name tag.” 店员们愣了一会儿立马互相笑着说:“Right! You are working here! Wearing the apron!”
这是一家以 LGBT 为主题的店。店里可以的颜色很鲜明，而且“彩虹”主题出现在很多地方。

六月是同志骄傲月。起源是石墙事件。所以这家店为了响应骄傲月，推出了 Proud Unicorn。味道很一般，只是一般的牛奶咖啡上加了奶油和 sprinkles。

Space Needle 对我来说并没有什么特别出彩的地方。只是一个很高的可以看到西雅图全貌的建筑。我还挺害怕高处的。观光台的边缘有着玻璃墙和玻璃凳，所以许多游客趴在墙上，站在凳子上留影。二楼是旋转的观光区，而且加下的地板时透明的。我鼓起了所有能找到的勇气走在上面，完全不敢往下看，假装自己走在平地上。

旁边的 Chihuly Garden & Glass 更加有意思一点。Chihuly 是一位玻璃艺术家，擅长用玻璃制作艺术品。我喜欢 Macchia 这批作品。意大利语中“斑点”的意思。Macchia 的做法是先烧融一层玻璃，然后在表面加上一层白色玻璃，烧融后再加另一层其它颜色的玻璃。也可以烧制三层以上。最后的结果是中间的白色玻璃做出了一种漂亮的云朵效果。这批作品的原名是 “The Uglies”，因为 Chihuly 的妈妈觉得这些作品很丑。后来他问了自己的意大利朋友“斑点”的意大利语才用了新名字 “Macchia”。

中午去了 Pike Place Market 吃午饭。第一家星巴克外面有着老长的一条队伍。我去了人比较少的 Pike Place Chowder。杨红樱在假小子戴安中描述巧达海鲜汤是装在面包碗里的。吃法是先喝一点汤，再撕下一点面包吃，如此重复，直到吃完。现实是面包碗并不深，而且外皮比较结实，所以很难撕下来。吃到最后我的左手手指蘸满了汤汁。

回去的时候遇见了一个奇怪的人。他在我吃完饭后就像选中了我一样跟在我后面。我停下里后他笑着向我搭话。他说自己这一天过的很不好，能不能让我请他喝杯酒。我刚吃完饭，脑子转不过来，居然说了好。但是三秒钟后又觉得这事不对。这是一个邋邋遢遢满脸胡渣的男人，而且他明显是我在他旁边路过的时候才开始跟着我的。而且他想去两个 block 外的地方买酒。我借口自己有急事处理，不能多待，在两步以外的地方给他买了一杯酒就走了。
不过我为什么要花钱买酒呢？

Brittany&Daniel 提供了一张他们自己的推荐卡，里面给出了一系列的餐厅和观光景点。附近刚好有两个：Cupcake Royale 和 Northwest Art and Frame.
早上去的 Biscuit Bitch 本身就是一家 LGBT 主题的店，但是很多一般的店铺都挂起了彩虹旗。Cupcake Royale 刚好就推出了 The Gay:

Forward III

Posted on 2019-06-18 Edited on 2019-06-20

在火车上半躺着睡觉并不舒服，尤其是在没有毯子或者外套的情况下。以前做飞机最长也就是 14 个小时，这回是 36 个小时的车程。窗外的风景也并不是那么值得，对我来说。后半夜的时候路过一个灯火通明的城市，每一栋建筑都闪着黄色的光。
我在凌晨四点的时候醒来，窗外漆黑一片，觉得冷。在下层车厢发现了一间比较宽敞的卫生间。趁着没人立刻拿上牙刷和毛巾洗漱。自从留了长发之后就必须要每天洗漱，不然容易油腻和发痒。想剪头发，剪一个 dmc5 Nero 的短发。如果还打算搭乘芝加哥的 Zephyr，我一定要选择 sleeper cart。在日出的时候路过森林和覆盖着雪的山。森林的边缘可以看到人为修建出来的草地和栏栅，旁边有两座房子。再过去一点可以看到更多的房子，大多带着庭院。我猜这是乡下，因为没有看到许多娱乐的地方，房子外停的车也是很老的款式。
餐车里要和陌生人坐在一起，和人搭话。我现在不喜欢做这些事情，强迫笑脸很累。阳光透过车窗照射在脸上很舒服。实际上早晨和黄昏的阳光都一样，如果不是因为刷过牙，嘴里有薄荷的清凉感觉，我都察觉不出区别。

到了下午的时候，有一位老人坐在了我的身边。中间有一段时间他很疑惑列车为什么走了错误的方向：原本应该是朝着北方，但是却朝着南方前进。我完全没有注意到这个变化。他在临近西雅图的时候问了列车长这个问题，列车长却说“why don’t you take off the train and sleep in a Motel, and face the right direction tomorrow?” 他回到座位后自言自语。“He has no time for me. A smarter person can answer my question. Maybe the gal working in cafe.” 我不知道该说什么。过了一会儿他又问了另一位列车长。原来有另一辆火车要往东西方向走，所以为了避免交叉，所以绕了个路。

Brittany&Daniel 的房间更加像个卧室。墙壁上挂着艺术画和一副世界地图。地图上插着大头针，代表着曾经来过的客人的家乡。大部分都是北美和欧洲，唯一没有的大头针的是南美。南美是最湿润的大洲，我记得很清楚，初中地理学过。

Forward II

Posted on 2019-06-16 Edited on 2019-06-20

房间里是没有大灯光的，唯一光源是一盏小台灯和窗外的光。窗外是一个天井，边缘有一个麦当劳的招牌。关灯后招牌灯光透过窗户反而变得和月光一样。昨晚在半梦半醒之间看见窗户外灯光变幻，从蓝色变成黄色，好像月亮在改变颜色。

这是一个墨西哥人和非裔美国人的社区。很少看到高加索人，亚裔也很少见。从商店招牌到涂鸦创作，墙上到处都是颜料。对我来说许多涂鸦都是我完全不能理解的图画。街上并不是特别干净。

到了车站后就开始预见各种各样的人了。Business Class 里面的老年人居多。我猜年轻人们大多不愿意在这上面多花钱。我见到了两个漂亮的小女孩。其中一个把头发染成了很难看的绿色。

我还以为自己会特别努力的编辑视频，但是在火车上并不是什么好的工作场合。
刚刚开出城区的时候，两边的土丘上有着许多的岩石，上面有着喷绘涂鸦。过一段时间后左手边的窗户就能看见海了，而右手边的的窗户外有很多小山坡。上面长有一大片开着黄色小花的绿色植物，黄色和绿色融在一起非常漂亮。在国内的时候，即使在郊区，我看到的大多是田地，很少看到这么漂亮的自然景象。打个比方，就像是装了美化的 Skyrim 中的山丘一样。

在 Dinning Cart 上遇见了三个老人。应该说是年长的人。其中一个是小学老师，她很可惜许多家里贫困的学生的家长更愿意把钱花在日常上而不是在教育上。“Only ten percent of them realized that education is key of success.” 她的丈夫是一位在电影中经常出现的中产阶级白人。他在自己的胸口别了一根圆珠笔。另一位女士的 sister 于 97 岁高龄逝世，她要回去处理相关事宜。她的丈夫是来自加拿大的苏格兰人，因此她自嘲自己有点吝啬。中午吃的是素食汉堡，比起 Pippin 的汉堡差远了。
实际上车厢很空旷，乘客们比我想象的要少很多。但是列车长在发车前说今天的人很多。人都去哪里了？
我穿了一件九又四分之三月台的T恤，没想到被很多人看出，纷纷指出这是一件很适合火车的T恤。但我在挑选T恤的时候并没有想这么多。
这趟旅程我准备的并不充分。实现也没有看完攻略，比如我没有带稍微厚实一点的衣服或者一件毯子。Coast Starlight 的商务舱并不提供 amenties kit，只有一瓶水和一张餐车优惠券。餐车的价钱稍微贵了一点，还在我的可接受范围内。

坐在我前面的是一个优雅的老人。他穿着白色衬衫，给自己倒了一杯酒。

天黑的时候，周围的的房子亮起了灯光。我却觉得有点瘆人。就像随时会发生什么可怕的事情一样。到了半夜，火车在桥上摇摇晃晃的行驶，从窗户向外望去能看到另一座大桥，桥上的灯光落在水面上发光。河水像是黑色的液体在翻滚；又像是有无数细小的生物在地面上爬行，它们的身体反射灯光造成了水面的假象。

我昨天晚上梦见下暴雨，自己躺在草地上抱成一团试图睡觉。有一个巨大的闪光灯照着我，一个男声疑惑地问我在做什么。我回答说我觉得冷。

还真是挺冷的。

  2. Move your dominant hand to strike your other hand in the palm with four fingers as illustrated here.

It’s impressive that XIV dev team made so much effort in details to represent a realist fantasy world.

Binary, Decimal, Graycode

Posted on 2019-05-26 Edited on 2020-11-29

Gray code

Gray code can reduce the errors of changing bits (or switches). It only needs to change 1 bit at a time to increments the decimal number.

Example 1

decimal binary gray code
5 101 111
6 110 101

From 5 to 6, binary has two bits changed, while gray code only changed 1 bit.

XOR is the method of converting binary to gray code. First we copy the most significant bit in binary to gray code’s 1st. Starting from 2nd most significant bit, we use XOR on the current bit and the previous bit.

Example 2

binary gray code
1, 0, 1 1, ?, ?
1, 0, 1 1, XOR(0, 1) = 1, ?
1, 0, 1 1, 1, XOR(1, 0) = 1
1, 0, 1 1, 1, 1
1, 1, 0 1, ?, ?
1, 1, 0 1, XOR(1, 1) = 0, ?
1, 1, 0 1, 0, XOR(1, 0) = 1
1, 1, 0 1, 0, 1

Binary and Decimal

To convert binary to decimal, simply multiply each bit by $2^{n}$ where n is the order of the position starting from the least significant number as 0.

Example 1

Binary: 1011
Decimal: $1 \cdot 2^0 + 1 \cdot 2^1 + 0 \cdot 2^2 + 1 \cdot 2^3 = 1 + 2 + 0 + 8 = 11$

To convert decimal to binary, simply divide the decimal by 2 and takes the remainder. Please note that the result starts from the least significant number.

Example 2

decimal binary
11 ?
11/2 = 5, r:1 1
5/2 = 2, r:1 1
2/2 = 1, r:0 0
1/2 = 0, r: 1 1
11 1011

Rotation Matrix

Posted on 2019-05-14 Edited on 2020-11-29

$P = (x, y) = (1, 0)$ $P’ = (x’, y’ )$

$\cos{\theta} = \frac{x’}{x}$ $\sin{\theta} = \frac{y’}{x}$

therefore:

$x’ = \cos{\theta} \cdot x$ $y’ = \sin{\theta} \cdot x$

similarly:

$Q = (x, y) = (0, 1)$ $ Q’ = (x’, y’ )$

$\cos{\theta} = \frac{y’}{y}$ $\sin{\theta} = \frac{-x’}{y}$

therefore:

$x’ = -\sin{\theta} \cdot y$ $y’ = \cos{\theta} \cdot y$

add them together:

$\begin{bmatrix}
\cos{\theta} & -\sin{\theta} \\
\sin{\theta} & \cos{\theta} \\
\end{bmatrix}$, it rotates counter-clockwise

$\begin{bmatrix}
\cos{\theta} & \sin{\theta} \\
-\sin{\theta} & \cos{\theta} \\
\end{bmatrix}$, it rotates clockwise

Jibber Jabber 3

Posted on 2019-04-19 Edited on 2019-04-23

iPad

iPad Pro 新出了一系列的 how-to 广告和 Life on iPad。看完之后产生了自己的生活也像广告中的那样精彩的幻觉：到处旅行，制作音乐和视频，能策划婚礼和生意，时不时和两三个人视频通话，还能单手旋转带着 Apple Pencil 的 iPad 并且不让任何部件掉在地上。
实际上我的生活比这单调的多。iPad 派不上什么用场。

溪谷恩说 Surface 指不定就行。
看了之后觉得 Surface 的广告不行。
我觉得我就不可能会突然拿着平板开始和一群不认识的人在街上或者办公室跳街舞。太羞耻了。平板还可能会掉进喷泉里或者摔在地板上。

撇开 Surface 不说，iPad 实际上就是一个能用 Apple Pencil 的大屏幕 iPhone。iOS 和非鼠标操作极大地限制了 iPad 在专业领域能发挥的作用。在 Surface 和 iPad 的广告中都在使劲强调“笔”这个配件。设计师，艺术家，文字相关的工作，还有学生，可能会从中受益。但是我不能用它编译代码或者制作 3D 模型的话，iPad 对我来说就是个玩具。虽然 Apple 一直宣传 iPad 是电脑的替代品，可是它的使用场景实际上是非常有限的。
没用过 Surface。考虑到它的系统是 Windows，我感觉它站在笔记本和台式机的中间某个位置。

Calvin Ball

Calvin Ball is a game. Go back to the magic circle theory and ritual, wearing Calvin mask, which is the only fixed rule, is the ritual. After players enter the magic circle, they begin to make rules against each other. It’s hard to tell who wins the game at the end, but I think competition is not the core of Calvin Ball. It is an exercise of creativity.
It is like “Yes And.” Players say “yes and…” to build upon each other’s statements until one of them says “No.” The improvisation is the mechanic.

Sherlock and Watson

No, it’s not the literature or the media.
Before Spotlight, Mac OS X used to have a similar tool “Sherlock” that allows users to search files and contacts. Then “Watson”, a $30 application as Sherlock’s companion, was born. It provides more internet functionality. Users can calculate exchanging rates, look up movie schedules, browse internet directory and more.
Then Sherlock 3.0 came with the same functions, like what they did to F.lux. Hence the “Sherlock” became the term describing such behavior.
Now it’s rumored that iPad will enable native support of external display for Mac. I can see many apps that provide the same function will get hurt.

迷宫饭，扇西日记

通篇都是网络流行的日式小说的口吻。完全不像一个西方奇幻文学中的矮人会说的话。
据我观察，在宅文化圈待过一段时间的人更容易使用这种口吻。遣词用句营造出一种“日漫少女”的感觉。

一些必备字词：了，呢，家伙，担心，果然。
例句：果然是让人担心的家伙，但没有发生什么事情真是太好了呢。

更简单的用法是在句子的末尾加上“的呢”或者“了呢”就可以。

例句：
巴金斯先生带回来了许多财宝。
巴金斯先生带回来许多财宝了呢。

凯兰崔尔的戒指很漂亮。

方便面 3

Posted on 2019-04-19 Edited on 2019-04-23

又开始迷上方便面了。上次吃日本的日清拉面，因为太好吃就吃到想吐。
这次吃韩国的。

Hot Chicken Flavor Ramen

就是那个有的人说辣，有的人说不辣的火鸡面。对于我这种不能够吃辣的人来说，肯定是辣的。
每一包有一个面饼，一包芝麻和蔬菜，一包辣酱。
辣酱放进去后很快就出现了那种红油一般的颜色。已经可以感受到辣味。我在煮面的时候室友刚好走进厨房，立刻打了个喷嚏，也不知道是不是因为辣。尝起来算是比较辣的面。我的感觉是在甜椒之上，干辣椒之下。这个排名没什么用处，但我这几年来吃过的辣椒也就这两种，所以也做不出什么比较。
吃的时候请用遮挡物围住衣服。有先见之明的我成功保护了我的衣服。

Nongshim Chapagetti

谷歌告诉我这是炸酱面。
每一包有一个面饼，一包蔬菜，一包粉料，一包油。
韩国的面包就比较厚实。每一根面条都是比日清或者康师傅的面条的要粗一些。
煮起来没什么味，吃起来也没什么味。但是很魔性的还想吃？就莫名其妙地还想吃。吃了两天后终于腻了。

Nongshim Neoguri

这个是辣的海鲜面。
每一包有一块面饼，一包粉料，一包蔬菜。
煮面的时候香味四溢。辣味比火鸡面弱很多，在我的可接受范围中。面条很有嚼劲。
我觉得比前面两种都好吃。

Shin Noodles

辛拉面。
每一包有一块面饼，一包粉料，一包蔬菜。
很 Neoguri 很类似，但是有更强的刺激性气味。并不是贬义。
就一般的辣味面条。

Samyang Ramen

三养拉面。
每一包有一块面饼，一包粉料，一包蔬菜。
这个是属于那种长了后会觉得“这就是方便面”的方便面。味道比辛拉面好，和 neoguri 是一个级别。面条比其它的都要细，也没什么嚼头。汤汁很好喝，而且是包装袋上的那种橘红色，不像火鸡面的汤就是辣油的颜色。
