Minimax Search

Posted on 2018-12-02 Edited on 2018-12-03

Games as search

“max” and “min” two players.
Max moves frist in the game. The game take turns until it’s over. Max uses search tree to determine “best” next move.

Definitions

  • initial state: set-up defined by rules
  • player(s): which player has the move in state s
  • actions(s): set of legal moves in state
  • results(s, a): transition model defines result of a move
  • terminal_test(s): true if the game is finished; false otherwise
  • utility(s, p): the numerical value of terminal state s for player p
    • win: 1, lose :-1, draw: 0

Procedure

  1. generate the whole game tree to leaves
  2. apply utility function to leaves
  3. back-up values from leaves toward the root:
    1. Max node computes the max of its child values
    2. Min node computes the min of its chidl values
  4. at root: choose move leading to the child of highest value

Alpha-Beta Pruning in Minimax

If a position is provably bad, it’s no use to search the rest.
If the adversary can force a bad position, it’s no use to search the rest because the adversary won’t let me achieve.

Begin at root (MAX), set:
alpha = -infty for Max
beta = infty for Min

for each child node:
inherit alpha and beta, check its children:
for each child:
assign its value to beta, if beta < alpha, prune it: terminate this child’s search
update parent’s alpha/beta: alpha = child’s beta, beta = child’s alpha
update root’s alpha beta

Why?

Max only chooses the maximum number, Min only chooses the minimum number.
Therefore, when beta(min) < alpha(max), Min will choose the beta’s value. There’s no use to search the rest children because no matter beta can be smaller or bigger, the situation won’t be better. Min only chooses smaller number. We already know beta is small.

Omensight

Posted on 2018-10-24 Edited on 2019-04-23

Intro

Definitive Version 打了个八折，没忍住就买了。我对靠近 ACT 类型的游戏兴趣还是蛮大的。而且许多游戏网站的评分也挺高，感觉质量应该不会差到哪里去。

剧情

大量剧透：

开头 Witch 的解说，Pygarians 和 Rodentians 之间的战争终于到了最后阶段。女祭司 Vera 早已被谋杀。Pygarians 的皇帝Indrik 准备好要化为 Firebird 保护世界。被囚禁在帝国监狱的 Rodentians 反抗组织的头领 Ratika 准备逃狱。Ludomir 被鼓动去正面对抗 Indrik。帝国将军 Draga 正在准备最后一场战斗。到了午夜，Indrik，Ludomir，以及 Draga 被杀，Ratika 则是化身 Deathless。Voden 突破封印，世界末日。
而玩家则是扮演 Harbinger，被 Witch 引导，不断的穿越回最后一天的清晨，与不同的角色一起，找出是谁杀以及为什么杀了 Vera，并阻止 Voden 带来的末日。

结局算是一个 BE。即使拯救了世界，Ratika 还是会变为 Deathless 杀死 Draga 成为新的统治者。追求 Voden 的士兵们还是会杀了皇帝。就算 Ludomir 活了下来，最终也会因为“醉酒”而发生一场所谓的“意外”而死。
在我个人看来，这个结局其实也算合理。整个游戏吸引人的地方在于一次又一次的穿越回去，探索不同的地区，与不同的角色一起发掘真相。但是每一次只能与一个角色一起行动，而且无法影响大局。换句话说，与 Draga 一起前往神殿就无法阻止 Ludomir 被杀；与 Ludomir 一起去找 Kladen，Ratika 还是会成为 Deathless；质问 Ratika 后一起寻找 Vera 留下的线索阻止不了 Indrik 被背叛；而与 Indrik 前往神殿，Draga 也会进攻 Rodentia。
因此，从一开始就注定了 Harbringer 只能阻止末日，不能让一切都完美结局。

不过 Definitive Version 更新了一个新的结局。达成条件是在收集了所有的路线和线索之后，Vera 路线会开启，Harbinger 会穿越到 Vera 被杀之前，完成就可以达成 HE，大家相亲相爱和和美美。我觉得 HE 没有 BE 有深度。

操作

现在我特别喜欢用手柄操作 ACT 类型的游戏，觉得比键鼠顺手多了。但还是掩盖不了手残。
Harbinger 使用剑来攻击，用 token 来增加伤害，防御，其它能力。每次升级都能获得一种特殊能力或者特殊能力的升级，包括：主动开启子弹时间，抓取物品和敌人，冲刺，以及必杀。
Combo 比较少，基本上就是轻/重攻击的三连组合。打击感无限接近与零，和 FFXIV 差不多吧。闪避后进入子弹时间这种设定在 Bayonetta 之后我还没有发现更出彩的。跳跃攻击基本上只用来对付浮空的 Cipher 和 Indrik，十分鸡肋。
老鼠类型的敌人投过来的炸弹是真恶心，很难预判所以大部分时间都是被打。而且炸弹会造成一个小型的持续的危险区域，每次都死在这个上面。一部分 Pygarians 士兵会开启绝对防御的姿态，必须要用抓取来破解，这个我特别喜欢，玩家需要思考一下而不是硬碰硬。
很多战斗都可以通过利用周围物品或者建筑来造成伤害。比如说，首先进入子弹时间，将炸药扔向石柱，石柱之后就会倒下压在敌人身上。或者让 Ludomir 重击地面，让一大排石柱倒下。或者启动敌人的喷火机关来烧伤敌人。我觉得这是比较聪明的设计，给玩家更多空间去战斗。
但是为什么这种攻击会算成 Stylish Attack？Stylish Attack 不应是 combo 吗？

最终 boss 战因为 Kladen 可以无限自动充能，所以可以很赖皮的进入子弹时间然后使用必杀。基本上就是稳赢。没什么特别大的难度。

一些木桶或者壶藏着苹果可以用来恢复，似乎苹果的数量取决于收到的伤害？因为有时候苹果特别多，而我的 HP 已经差不多要没了。大家都喜欢苹果啊……塞尔达也是……我打工的餐厅也是。

另外，如果打碎一些物品来获得苹果或者收集 token，所有同行的角色都可能会吐槽这种举动。“你到底在做什么？” “你的剑出问题了吗？” “破坏东西是你的爱好吗？” “留着力气打碎敌人吧。” “等战争结束了你想炸什么都可以。”

路线

这个！
就是解密！
整个游戏内有三个 Omensight，Vera 路线的不算。所谓 Omensight 就是 Harbinger 发现了重要线索后的 vision。Harbinger 可以选择让角色看见 vision 来告诉他们一部分真相，以此进入不同的支线。
我个人比较喜欢 Ratika 和 Draga 的路线。Ratika 是一只唠唠叨叨的老鼠吟游诗人，用乐器 Balalaika 和音乐来施展魔法。她知道一部分真相，察觉了更多的秘密，却还是选择了错误的路线成为了 Deathless。Draga 则是皇帝忠诚的猫将军，但是非常理性。在她知道皇帝可能杀死了 Vera 后选择去质问皇帝，而不是拒绝相信。另外她的声音也挺好听的。

醉熊 Ludomir 的线路基本上都是“我好生气，我要杀了你！”。求求你冷静一下吧。

实际上路线分支不是很多，因为一共只有四个角色路线，三个 Omensight，而且每一个 Omensight 都要经历至少两到三个角色才能够进入下一个阶段。所以玩到最后，除非特别幸运一下就能找到关键地方或者开启了向导模式，玩家在最后都会收集到大部分的路线。

音乐

完全不出彩，出了 Ratika 的那首歌以外，完全记不起来有什么“音乐”的地方，大部分都是两三秒的人声？
真的，我完全想不起来了。

Conclusion

Omensight 有趣的地方在于解密和重复不同的路线，以及利用环境来战斗。但是其他的地方完全没有看点。通关并且完成 Vera 路线后就没有重玩的想法了。

关于 SCH 的技能命名

Posted on 2018-10-08 Edited on 2019-04-23

Scholar/SCH/学者，为 FFXIV 的一种基于 Arcanist/ACN/秘术师的职业。

下面列出四个技能的英文名和中文名，中文名的翻译基于日文名就用中文名来比较：

  • Adloquim 鼓舞激励之策

  • Succor 士气高扬之策

  • Sacred Soil 野战治疗阵

  • Lustrate 生命活性法

秘术师的任务链一直在强调用 strategy/策略，分析局势，以此来获得胜利。学者也是古代城市 Nym 中海军里运用战略战术的人。从名字来看，中文版/日文版都比英文版的更加具有“学者”风范。所以为什么在 localization 的过程中不使用”method“或者”plan“之一类的单词呢？

因为：

  • Adloquim 是拉丁语，意为“reasrruing word，word of encouragement“，也就是“鼓励的话语”的意思。
  • Succor 意为 “give assistance, aid to”，也就是“给予帮助”。
  • Lustrate 意为 ”purify by expiatory sacrefice, ceremony, or some ritual action”，“通过奉献牺牲或者仪式来达到净化的目的“

以上三种翻译都可以用一些古老或者简单的单词来表达相同的意思，而且也不失”学者“的感觉。
但是 Scared Soil 我想不出来有什么特别的意思。而且我认为完全可以和 WHM/白魔导士的 Asylum/庇护所 互相交换一下。因为根据效果和名字来看：

  • Sacred Soil “神圣土地“ 制造一个半球形的领域，站在内部的玩家收到的伤害减少伤害。

  • Asylum/庇护所 制造一个半球形领域，站在内部的玩家能够逐渐恢复 HP。

白魔导士的技能大多都和”神圣“有关，比如 Benediction/天赐祝福，Holy/神圣，Divine Benison/神祝祷。Sacred Soil 更像是白魔导士的技能。而 Asylum/庇护所 意为“提供保护”，也可以理解为减少伤害。

学者的其他技能名字也很有意思：

  • Excogitation/深谋远虑之策

    深谋远虑是指提前就想好了对策，excogitation 意为 “think out, plan, or devise“，而且它的词根起源也是拉丁文“excogitate-”，也表示 “found by process of thought“。都很符合翻译。

  • Broil/气炎法

    这个特别有意思。気炎 在日文中可以表示“激烈的讨论”。Broil 的一种古老用法也是表示争吵 ”quarrel“。
    FFXIV 的英文魔法技能没有使用 FF 传统的 raga 命名，而是增加数字尾缀。比如 Cure 的增强是 Cure II，再增强就是 Cure III。Broil 的增强版就是 ”Broil II“。但是日文版和中文版的反而是 “魔炎法”。我不知道“魔炎”在日文中有没有其它意思，但是比起 “気炎” 类似于双关的用法，“魔炎” 稍显逊色。
    这个名字是指学者都是吵架王吗？

  • Indomitability/不屈不挠之策

    这个就表示 “impossible to subdue or defeat”，“无法被镇压，克制或打败”，差不多就是一个意思。

  • Emergency Tactics/应急战术，Deployment Tactics/展开战术，Chain Strategem/连环计

    这三个基本上就是对应着翻译，就是字面的意思，没有像前面的英文名一样简练，有内涵。

顺带，FF 系列的 raga 命名法。

-ra, -ga 为基础的三级命名。第四级为 -ja。据说还有第五级的 -za。
举例：
火魔法：Fire, Fira, Firaga, Fiaraja.
冰魔法：Bizzard, Blizzara, Blizzaga, Blizzarja。

真是个方便的命名系统。

闪亮闪亮的古剑奇谭网络版

Posted on 2018-10-02 Edited on 2019-04-23

Intro

就是想玩一下。
现在天朝实行了实名制，网元圣唐的账号注册需要手机号码或者 ID。据称 Police 部门还会检查。但是这个邮箱注册只需要 ID 和真实姓名。随机生成了一个天朝 ID，随机取名”李渊“。每次登陆都要验证一遍这个虚拟身份。
说起来我还看过了一篇李渊 x 李世明 x 李建成的现代背景的没有下限的文。特别低俗。

安装

下载器似乎用了 P2P，虽然这不是非法的事情，但是我心里有道坎过不去。用了 http 下载。
但是安装路径如果有中文就会 launch 失败。这个过分了。压缩包里就是中文，安装说明却没有提醒。这不就是当场劝退么？

妙法

我就觉得妙法的衣服挺好看的，就选了这个宗门。随后游戏一个小时内就给了我一套我很不喜欢的清晖。前 14 级就有点像 XIV 4.0 之前的 THM，是个法师。可以积累上限为六个的两种“音”，然后用特别技能消耗“音”来增加伤害。Astral Fire & Umbral Ice 与 Fire & Blizzard。
任务流程非常的 linear。各种意义上的 linear。接/做/交任务都是直线形的。我可以一直往前走，不用回头。好处是特别方便玩家，坏处是我根本不知道我在哪里，完全没有熟悉地形，就跟着任务图标玩前走。

经常出现人物突然后跳，疑似因为 latency 和 client prediction 导致的 convergence。但是为什么会有敌人会突然一个大跳就到我身边？

有一个女声啰哩啰嗦的指导我怎么控制角色。我不想听，你给我文字解说就行。
另外不管怎么样配音都有一种在听动画片的配音的感觉。就是一种字正腔圆，没有口音，中规中矩，稍微端着，像又不像是普通地在说话的感觉。

剧情到了后面就被配音弄烦了，blah blah blah 剑心 blah blah blah 变身反派 blah blah blah “以后多的是机会收拾他们” blah blah 我的情郎走了我好伤心 blah blah 杀杀杀 blah blah 你居然 blah blah 我忘了剧情。
能够说出”以后多的是机会收拾他们“这种话的反派就是个傻逼。斩草不除根，春风吹又生。这么简单的道理都不懂还做什么反派。看见正派受伤了怎么办？上去就一刀灭了免留祸患。看见一个突然就出名的无名小辈倒在你面前怎么办？弄死免得以后成为英雄来杀你。
剧情里面的某一个拿着长杖的 NPC，忘了是谁，特别像 XIV 中 CNJ 和 ACN 的混合体。

物理效果有点过分。比如衣摆的摆动，就是稍微动一下就会变得像油布一样，还有特别顺畅的摆动，看上去感觉就不太真实。像以前教授给我布置的 WebGL 作业中的布料。
色彩是比较明亮的。每个人都面色红润，光鲜亮丽。每个地方都光线充足。方便截图？

捏脸

我一直不明白为什么天朝这么注重捏脸呢？是因为天朝玩家心中 customization 占的比例特别大吗？
装备还能一套一套的保存？摄形？我选了一个自己喜欢的发型，为什么一戴上头部装备就会改变发型？这样选择发型有何意义？
glamour 是如此重要的一个题目。

喷气式轻功

据说是现在的仙侠/武侠游戏的标配，三段跳。不能充值的我只能用二段跳。就特别酷炫。比来须苍真的一飞冲天的铁头功还酷。但是不如 Bayonetta 的 Witch Time 酷。Witch Time 这套系统真是能玩到死。
B 站上看到了特别有技巧的轻功视频。就非常棒。然后还有需要跳到最高处才能接近的商人。

轻功怎么看怎么酷。

喷气式。

UI

看上去每个宗派都有一个类似于 Job Gauge 的 UI。比如妙法的就是六根琴弦，三根代表”冰“，三根代表”火“。
整体上并没有很多无关信息。左边有一个类似于 quick access 的列表，会列出最近获得的游戏内容。可以快速装备新获得的装备就特别棒。没有出现那种攻击伤害“999”的数字真是太好了。左下角的聊天频道无论是在天朝的白天还是夜晚都是在骂人以及嘲笑斩风的绿色设计，这个关于绿色的吐槽真是被妖魔化了…… 可能我上线的时间不太好。
每次 load 场景的时候都会出现一种类似于将渲染视觉化的效果，就特别酷。我还挺喜欢的。

Conclusion

这个轻功还真好玩。如果轻功可以加入游戏战斗 机制中我就觉得特别有意思。
景色挺好看的。但是除开一小部分敌人，每个人物相貌端正，像是日本少女漫画中背景是花团锦簇并且贴上了各式各样闪亮网点的人物。就总觉得这群人……好烦哦。
就是一个 14 级的小人物，没有去挖宝或者下副本，只能说一说一小部分游戏体验。

若是有 hack&slash 或者传统 ACT 这样的仙侠/武侠游戏岂不美哉。目前只看见《血雨》是这种类型。就真的是修习功法了，因为要记住 combo。

游戏内角色名是“李九歌”。妙法拿着一把琴，游戏里面又是有什么大司命，浮空飞行的仙法什么的，所以用了屈原的“九歌”。
